### Method 3: Create from Scratch

1. **Start Fresh**
   ```
   Navigate to: https://your-domain.com/lab
   ```

2. **Describe Your Game**
   - Use the AI prompt box at the top
   - Example: "A top-down adventure where Link explores dungeons"
   - Click "Generate My Game"
   - AI fills in all the fields automatically!

3. **Generate Custom Sprites (Optional)**
   - Scroll to sprite upload section
   - Click "Generate with AI"
   - Describe your character (e.g., "heroic knight with shining armor")
   - Choose an art style (Pixel Art, Cartoon, etc.)
   - Click "Generate Sprite"
   - Preview and use it!

4. **Configure Settings**
   - Choose game genre (Platformer/Adventure/Puzzle)
   - Select programming language (Python/JavaScript)
   - Pick mood (Hopeful/Gritty/Heroic)
   - Set difficulty (Rookie/Veteran/Nightmare)

5. **Build**
   - Sign in if you haven't already
   - Click "Build & Publish Game"
   - Watch the real-time build progress
   - Play when complete!
